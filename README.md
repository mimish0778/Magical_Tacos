# タコ先生の8本足会議

ドスパラ主催 AI ハッカソン 制作作品

悩みや課題を入力すると、たこ先生が8本の足それぞれ異なる視点からアイデアを提案してくれる Web アプリです。

## 視点一覧

| 足 | 視点 |
|---|---|
| 1本目 | 逆転の発想 |
| 2本目 | 引き算の発想 |
| 3本目 | 他人の視点 |
| 4本目 | 時間軸をずらす |
| 5本目 | 遊び心の発想 |
| 6本目 | 環境を変える |
| 7本目 | 小さな一歩 |
| 8本目 | 誰かに頼る |

## 技術スタック

| 項目 | 内容 |
|---|---|
| フロントエンド | HTML / CSS / JavaScript（単一ファイル） |
| AI API | Gemini API |
| API プロキシ | Cloudflare Workers |

Gemini への直接アクセスを避けるため、Cloudflare Workers を API プロキシとして挟んでいます。
